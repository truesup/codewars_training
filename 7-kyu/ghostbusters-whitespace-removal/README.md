<h2><a href=https://www.codewars.com/kata/5668e3800636a6cd6a000018/train/javascript target="_blank">Ghostbusters (whitespace removal)</a></h2><h3>7 kyu</h3><p>Oh no! Ghosts have reportedly swarmed the city. It's your job to get rid of them and save the day!</p><p>In this kata, strings represent buildings while whitespaces within those strings represent ghosts.</p><p>So what are you waiting for? Return the building(string) without any ghosts(whitespaces)!</p><p>Example:</p><pre><code>"Sky scra per" -&gt; "Skyscraper"</code></pre><p>If the building contains no ghosts, return the string:</p><pre><code>"You just wanted my autograph didn't you?"</code></pre>
