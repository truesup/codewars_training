<h2><a href=https://www.codewars.com/kata/5637b03c6be7e01d99000046/train/javascript target="_blank">Password maker</a></h2><h3>7 kyu</h3><p>One suggestion to build a satisfactory password is to start with a memorable phrase or sentence and make a password by extracting the first letter of each word. </p><p>Even better is to replace some of those letters with numbers (e.g., the letter <code>O</code> can be replaced with the number <code>0</code>):</p><ul><li>instead of including <code>i</code> or <code>I</code> put the number <code>1</code> in the password;</li><li>instead of including <code>o</code> or <code>O</code> put the number <code>0</code> in the password;</li><li>instead of including <code>s</code> or <code>S</code> put the number <code>5</code> in the password.</li></ul><h2 id="examples">Examples:</h2><pre><code>"Give me liberty or give me death"  --&gt; "Gml0gmd""Keep Calm and Carry On"            --&gt; "KCaC0"</code></pre>
