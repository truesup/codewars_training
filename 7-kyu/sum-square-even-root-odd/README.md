<h2><a href=https://www.codewars.com/kata/5a4b16435f08299c7000274f/train/javascript target="_blank">Sum - Square Even, Root Odd</a></h2><h3>7 kyu</h3><p>Complete the function that takes a list of numbers (<code>nums</code>), as the only argument to the function. Take each number in the list and <em>square</em> it if it is even, or <em>square root</em> the number if it is odd. Take this new list and return the <em>sum</em> of it, rounded to two decimal places.</p><p>The list will never be empty and will only contain values that are greater than or equal to zero.</p><p>Good luck!</p>
